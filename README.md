Progetto di analisi di un dataset JSON della classe quarta superiore del corso informatico dell'istituto tecnico I.T.I.S. Castelli.
Lo scopo è analizzare il dataset scelto attraverso programmazione in linguaggio Python, ad esempio per esplorarne la struttura o per filtrarne i dati. 
Il dataset JSON scelto è il catalogo di giochi Playstation 4 e Playstation 5 ed è stato ricavato da: "https://www.kaggle.com/datasets/evgeny1928/playstation-games-info/data?select=output.json"
